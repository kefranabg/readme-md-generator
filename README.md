> Generates beautiful README files from git config & package.json infos

## Installl

With npm :

```sh
npm i -g readme-md-generator
```

With yarn :

```sh
yarn global add readme-md-generator
```

## Usage

Just run the following command and answer questions :

```sh
readme
```
